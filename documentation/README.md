# TP3 - Redes Neuronales

Trabajo práctico de la materia I302 - Aprendizaje Automático y Aprendizaje Profundo.

## Objetivo

Implementar y evaluar redes neuronales para clasificación multiclase sobre EMNIST ByMerge.

## Estructura

- `Entrega_TP3.ipynb`: notebook principal de entrega.
- `src/`: implementación modular del proyecto.
- `data/`: datasets locales no versionados.
- `notebooks/`: notebooks trabajados.
- `results/`: tablasy métricas generadas.

## Ejecución

1. Colocar los archivos `X_images.npy` y `y_images.npy` en la carpeta `data/`.
2. Instalar dependencias:

```bash
pip install -r requirements.txt
```
